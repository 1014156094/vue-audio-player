# Vue音频播放器组件

## 概述
因为之前项目做过，为了方便以后使用，所以抽空弄成了组件。

## 安装项目
```
npm install
```

### 开始开发
```
npm run serve
```

### 生产打包
```
npm run build
```

## 注意
组件CSS使用 `less` 预编译语言

## 插槽
暂无

## Props
| 参数 | 说明 | 类型 | 默认值 |
| - | - | - | - |
| audio-url | 音频链接 | String | - |
| audio-name | 音频名称 | String | - |
